Este proyecto fue creado para la resolución de la actividad propuesta en la Unidad Formativa 2 del Módulo Profesional de Diseño de Interfaces Web dentro del Ciclo Formativo de Grado Superior de Desarrollo de Aplicaciones Web.

En su día, fue desarrollado dentro de las métricas establecidas, pero con la posibilidad de hacerlo propio en cuanto a la elección del tema.
En mi caso, escogí la Semana Santa de Lorca.

Para poder visualizar el proyecto en un navegador, se tendrá que descargar, o bien clonar el repositorio a través de GitHub Desktop.
